# Forkify App

App for searching recipes created with vanilla Javascript.

## Features

- Search recipes using Forkify API
- Create your own recipe (localstorage)
- Bookmark recipes

## Built With

- JavaScript (ES6)
- HTML
- CSS (Sass)
- Parcel
- NPM
